# add-config

## Install

```
$ npm install --global @barelyreaper/add-config
```

## Usage

```
$ add-config --help

  Usage
    $ add-config [options]

  Options
    -p, --path  path to consider as root
    -t, --type  type of config, currently supports [react|node]
  Examples
    $ add-config -p . --type=react
    Created!
```
